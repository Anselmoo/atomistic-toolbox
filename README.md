# Atomistic-Toolbox

## About

The Atomistic-Toolbox is collection of atomic parameter for spectroscopy and theory, which will be contiguously extend. It is built on [VUE 3](https://vuejs.org).

## Installation

```shell
npm install Anselmoo/Atomistic-Toolbox
```
